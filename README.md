### 🌌 Overview
Raw transaction data ke meaningful business strategy-te rupantor korai ei project-er core mission. 2.6 million-er beshi data-set handle kore amra consumer behavior analyze korechi, jeta e-commerce branding ebong inventory management-e critical role play kore.

---

## 💎 Project Highlights
* **Big Data Ready:** 2,600,000+ transaction rows processing capacity.
* **User-Centric Discovery:** Discovered 8 high-impact association rules from 91,619 unique users.
* **Multi-Category Reach:** Insights gathered across 13 diverse product categories.

## 📈 Association Rule Metrics
Amra nicher formula-gulo diye data-r logic validate korechi:
* **Support:** Itemset-er frequency calculate kore.
* **Confidence:** X kinar por Y kinar probability koto seta measure kore.
* **Lift:** Correlation check kore (Lift > 1 mane products-er moddhe strong relation ache).
