# Market-Basket-Analysis
Market Basket Analysis on 2.6M+ e-commerce transactions using Apriori Algorithm. Discovered 8 association rules from 91,619 users across 13 product categories. Built with Python, Pandas, MLxtend &amp; Matplotlib. Includes full report and presentation.

## 📈 Association Rule Metrics
Amra nicher statistical formula-gulo diye rules validate korechi:

* **Support:** Frequency of the itemset.
* **Confidence:** Probability of purchasing Y given X.
* **Lift:** The strength of the association (Lift > 1 means positive correlation).
